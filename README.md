# DevFest Workshop 日记

记录我的 Chrome DevTools MCP Server 学习之旅

## 项目简介

这是一个记录我参加 Google DevFest Workshop 学习经历的个人网站。在这次以 Chrome DevTools MCP Server 为主题的 Workshop 中，我深入学习了现代 Web 开发工具的使用和最佳实践。

## 网站内容

- **首页** - Workshop 总览和学习收获
- **GDG & DevFest** - 了解 Google Developer Groups 和 DevFest 活动
- **DevTools MCP** - 深入探讨 Chrome DevTools MCP Server
- **关于我** - 个人介绍和联系方式

## 特色功能

- 🎨 使用 VS.svg 作为背景，营造专业的开发环境氛围
- 🌟 凤心菜角色主题设计，增加个性化元素
- 📱 响应式设计，支持各种设备访问
- 🎯 清晰的导航结构和内容组织

## 技术栈

- HTML5 + CSS3
- 响应式设计
- GitHub Pages 自动部署

## 在线访问

网站已部署到 GitHub Pages，可以通过以下链接访问：
[https://nanamiaakari.github.io/devfest-workshop-diary/](https://nanamiaakari.github.io/devfest-workshop-diary/)

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/NanamiAkari/devfest-workshop-diary.git

# 进入项目目录
cd devfest-workshop-diary

# 启动本地服务器
python -m http.server 8000

# 在浏览器中访问 http://localhost:8000
```
